# Archived Open Notebook Bridge

This folder contains earlier experiments that wired CDISC/Ollama features directly into the core Open Notebook application tree.

Archived items here were previously placed inside:

- `api/`
- `commands/`
- `frontend/src/`

They are kept for reference only.

## Status

- Not part of the active Orange Pi deployment
- Not part of the preferred repository layout
- Safe to consult if you want to revisit a deeper integration later

The active implementation uses the separate mirrored API in `local-addons/cdisc/api/` instead.
